Worked Examples

Example 1: \(3a + 2b + 2a + b = 5a + 3b\)

Step 1 Lay out all terms as coloured chips.

All terms laid out in original order
aaa + bb + aa + b

Step 2 Re-arrange: group chips of the same colour together.

Grouped by colour (blue = a, orange = b)
aaaaa bbb

Step 3 Count each group to get the result.

Answer: \(5a + 3b\)

Example 2: \(7p - 3q - 2p + 5q\)

Step 1 Identify like terms: \(p\)-terms: \(7p\) and \(-2p\)   \(q\)-terms: \(-3q\) and \(+5q\)

Step 2 Rearrange: \(\bigl(\)\(7p - 2p\)\(\bigr) + \bigl(\)\(-3q + 5q\)\(\bigr)\)

Step 3 Collect: \(5p + 2q\)

Answer: \(5p + 2q\)

Hint: Only combine terms with the same variable. Constants (plain numbers) can also be collected together.
